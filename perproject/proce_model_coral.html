Procedural Modeling

These are class projects from my CGMA procedural modelling class. All of them are created in a learning environment with some enhancements and tuneups which are personal.

Coral Reef

Project Specifications

A System of Coral is created and painted on a bed of rocks. The coral distribution is determined by the painted surfaces on the rock. The system of rocks is created using the controls provided.
Every piece of this system is customizable.


the coral System has 4 types Digitate, Plates, Tubular and Branching.
Each system of coral has its own settings.
