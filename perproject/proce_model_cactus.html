Procedural Modeling - Cactus & Succulents

Project Specifications

Create a Cactus and a Succulent models with master controls to control the look and feel of the object / model.

Once the models were created each was used in 4-5 variation to put together this window box.

Cactus

The spines count, angle and cluster along with the twist, inner and out ridges etc, are designed to be modified from one cactus to the next with the controls provided.

Succulent

Here, the petals, shape, width depth and length can be modified to create a unique variation everytime.
