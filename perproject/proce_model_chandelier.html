Procedural Modeling

These are class projects from my CGMA procedural modelling class. All of them are created in a learning environment with some enhancements and tuneups which are personal.

Chandelier

Project Specifications

Create a procedural model of a chandelier such that the system can create other variation by modifiying a set of controllers exposed for the artist or user of the system.

The Chandelier System has the following settings that can be controlled and modified to create new versions.
1. Metal Rings - The rings on each tier<
2.Chain Link- The chains from the top support with interlocking system.
3.Supports - The beams in the center of the bottom tier
4.Upper - The Upper Ring system of the Chandelier with Droop
5.Lower - The lower rings with the hanging cystals

Each Ring system has modifications for strand height, space between the rings etc.
